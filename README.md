# Dynamic-causal-inference

Project developed by Manuel and Andrés for the course on causal inference and dynamic models.

The project analyzes the evolution of disappearances in Mexico and studies the potential impact of the implementation of the Guardia Nacional using causal inference methods such as Difference-in-Differences and Synthetic Control.

The repository includes:
- Data cleaning and preprocessing pipelines in R
- Construction of panel data structures
- Exploratory data analysis (EDA)
- Dynamic causal inference models
- Reproducible reports in R Markdown and LaTeX

Main tools:
- R
- tidyverse
- ggplot2
- fixest
- Synth
- R Markdown

Project structure:
- `data/`: raw and processed datasets
- `scripts/`: data processing and modeling scripts
- `docs/`: report, figures, and tables
- `src/`: auxiliary functions and utilities
